<Headings.h4 className="text-white bg-black inline-block px-4 py-2 rounded-full">restrictInputWithCounter</Headings.h4>

<Paragraph className="mt-2 text-gray-400">
The `restrictInputWithCounter` function serves as an alias to `restrictInputLengthWithCounter`, providing an alternative method to achieve the same functionality of restricting input length and updating a character counter.
</Paragraph>

<Headings.h6 className="mt-4">Example: Use `restrictInputWithCounter`</Headings.h6>
