<Headings.h4 className="text-white bg-black inline-block px-4 py-2 rounded-full">getPageUrl</Headings.h4>

<Paragraph className="mt-2 text-gray-400">
The `getPageUrl` function returns the complete URL of the current page, including the protocol (http/https), domain, port, path, and query parameters. This URL is retrieved from the `window.location.href` property.
</Paragraph>
