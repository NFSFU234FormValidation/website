<Headings.h4 className="text-white bg-black inline-block px-4 py-2 rounded-full">validateAllRadio</Headings.h4>

<Paragraph className="mt-2 text-gray-400">
The `validateAllRadio` function validates all radio button groups within a specified form. It returns an array of error messages (a nested <Link href="/docs/errorMessageInterface" className="text-blue-600 font-semibold" >ErrorMessageInterface</Link>) if any group is invalid, or a single instance of <Link href="/docs/errorMessageInterface" className="text-blue-600 font-semibold" >ErrorMessageInterface</Link> if all fields are valid.
</Paragraph>

  ]}
