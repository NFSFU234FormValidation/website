<Headings.h4 className="text-white bg-black inline-block px-4 py-2 rounded-full">redirect</Headings.h4>

<Paragraph className="mt-2 text-gray-400">
The `redirect` function navigates the browser to a specified URL. Optionally, you can provide a delay (in milliseconds) before the redirection occurs. If no delay is specified, the redirection happens immediately. If the URL is `null` or `false`, no redirection occurs.
</Paragraph>

<WorksIn visibleTypes={['reactJS', 'html5']} />

<ParameterList
  parameters={[
    {
      name: 'url',
      description: 'The URL to which the browser should navigate.',
      type: 'string | null | false',
      dataType: 'string',
      defaultValue: 'null',
      returnType: 'void'
    },
    {
      name: 'delay',
      description: 'Optional delay before redirection in milliseconds.',
      type: 'number',
      dataType: 'number',
      defaultValue: '0',
      returnType: 'void'
    }
  ]}
/>

<Headings.h5 className="mt-4">Usage</Headings.h5>

<Headings.h6 className="mt-4">Example: Redirect to a URL</Headings.h6>

<CodeBlock>
{`const url = "https://www.example.com";
nfsfu234FormValidation.redirect(url); // Redirects immediately to the specified URL

const delayedUrl = "https://www.example.com";
nfsfu234FormValidation.redirect(delayedUrl, 3000); // Redirects to the specified URL after 3 seconds
`}
</CodeBlock>
