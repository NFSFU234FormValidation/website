<Headings.h4 className="text-white bg-black inline-block px-4 py-2 rounded-full">loading</Headings.h4>

<Paragraph className="mt-2 text-gray-400">
The `loading` function changes the content of a button or input element to a specified message while providing loading feedback. This can be useful for indicating to users that an operation is in progress.
</Paragraph>

<WorksIn visibleTypes={['html5', 'reactJS']} />

<ParameterList
  parameters={[
    {
      name: 'message',
      description: 'The message to set as the content of the button or input element.',
      type: 'string',
      dataType: 'string',
      defaultValue: 'undefined'
    },
    {
      name: 'submitBtn',
      description: 'The button or input element to update. This can be a string (ID of the button), an HTMLElement, or null.',
      type: 'string | HTMLElement | null',
      dataType: 'mixed',
      defaultValue: 'null'
    },
    {
      name: 'form',
      description: 'The HTML form element or its ID from which to find the button if submitBtn is null. This can be a string (ID of the form) or an HTMLElement.',
      type: 'string | HTMLElement | null',
      dataType: 'mixed',
      defaultValue: 'null'
    }
  ]}
/>

<Headings.h5 className="mt-4">Usage</Headings.h5>

<Headings.h6 className="mt-4">Example: Update Button Content with Loading Message</Headings.h6>

<CodeBlock>
{`const formElement = document.getElementById('myFormId');
nfsfu234FormValidation.loading('Loading...', formElement.querySelector("button[type=submit]"));
`}
</CodeBlock>

<Headings.h6 className="mt-4">Example Form HTML</Headings.h6>

<CodeBlock>
{`<form id="myFormId">
  <input type="text" name="fieldName1" value="value1" data-attr-name="fieldName1" />
  <input type="text" name="fieldName2" value="value2" data-attr-name="fieldName2" />
  <button type="submit">Submit</button>
</form>`}
</CodeBlock>

<Headings.h6 className="mt-4">JavaScript to Update Button Content</Headings.h6>

<CodeBlock>
{`const formElement = document.getElementById('myFormId');
const buttonElement = formElement.querySelector("button[type=submit]");
nfsfu234FormValidation.loading('Loading...', buttonElement);
`}
</CodeBlock>

<Headings.h6 className="mt-4">Expected Output</Headings.h6>

<Paragraph className="mt-2 text-gray-400">
The content of the submit button within the form will be updated to display the message "Loading...".
</Paragraph>
