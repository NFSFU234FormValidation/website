<Paragraph className="mb-6">
    You can install the NFSFU234FormValidation Library via CDN links or npm or yarn

<Headings.h6> CDN (JsDelivr) </Headings.h6>

<Paragraph className="mb-2">
copy and place the code below into your 'head' tag of your HTML5 document
</Paragraph>

<CodeBlock>
  {`<script src="https://cdn.jsdelivr.net/npm/nfsfu234-form-validation@latest/dist/js/nfsfu234FormValidation.js"></script>
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/nfsfu234-form-validation@latest/dist/css/nfsfu234FormValidation.min.css" />`}
</CodeBlock>

<Headings.h6 className="uppercase">NPM</Headings.h6>

<Paragraph className="mb-2 uppercase">
To use this library in your NPM supported applications (React JS, Vue JS, etc.), simply copy the command below and paste it in your project's console to add NFSFU234FormValidation library and its capabilities to your project.
</Paragraph>

<CodeBlock>
npm install nfsfu234-form-validation
</CodeBlock>


<Headings.h6 className="uppercase">YARN</Headings.h6>

<Paragraph className="mb-2">
To use this library in your Yarn supported applications (React JS, Vue JS, etc.), simply copy the command below and paste it in your project's console to add NFSFU234FormValidation library and its capabilities to your project.
</Paragraph>

<CodeBlock>
yarn add nfsfu234-form-validation
</CodeBlock>

<Paragraph className="mb-2">
Based on your project's type, you can choose what ever method to integrate `NFSFU234FormValidation Library` into your respective project types. Now you can move on to how to initialize the `NFSFU234FormValidation Library`
</Paragraph>
