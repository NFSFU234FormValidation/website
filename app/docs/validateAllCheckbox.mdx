<Headings.h4 className="text-white bg-black inline-block px-4 py-2 rounded-full">validateAllCheckbox</Headings.h4>

<Paragraph className="mt-2 text-gray-400">
The `validateAllCheckbox` function validates all checkbox inputs within a specified form. It returns an array of error messages (a nested <Link href="/docs/errorMessageInterface" className="text-blue-600 font-semibold" >ErrorMessageInterface</Link>) if any checkbox is invalid, or true if all checkboxes are valid.
</Paragraph>
