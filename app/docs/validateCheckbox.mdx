<Headings.h4 className="text-white bg-black inline-block px-4 py-2 rounded-full">validateCheckbox</Headings.h4>

<Paragraph className="mt-2 text-gray-400">
The `validateCheckbox` function validates a specified checkbox input field. It checks if the checkbox is checked and returns an <Link href="/docs/errorMessageInterface" className="text-blue-600 font-semibold" >ErrorMessageInterface</Link>.
</Paragraph>

<WorksIn visibleTypes={['html5', 'reactJS']} />

<ParameterList
  parameters={[
    {
      name: 'checkboxInputField',

      dataType: 'HTMLElement',
      defaultValue: 'undefined',
      returnType: 'boolean | ErrorMessageInterface'
    },
    {
      name: 'options',
      description: 'Optional settings for validation such as custom error messages.',

    }
  ]}
/>

<Headings.h5 className="mt-4">Usage</Headings.h5>

<Headings.h6 className="mt-4">Example: Validate a Checkbox Input</Headings.h6>

<CodeBlock>
{`// Using with a checkbox input element
const checkboxElement = document.querySelector('input[type="checkbox"]');
const result = nfsfu234FormValidation.validateCheckbox(checkboxElement, {
  message: 'You must agree to the terms and conditions.'
});
console.log(result);
`}
</CodeBlock>
