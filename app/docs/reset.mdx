<Headings.h4 className="text-white bg-black inline-block px-4 py-2 rounded-full">reset</Headings.h4>

<Paragraph className="mt-2 text-gray-400">
The `reset` function resets the values of all form elements (inputs, textareas, and selects) within a specified form. If no form is provided, it defaults to the document's body. It can handle form elements specified by an ID, an HTMLFormElement, or an HTMLDivElement.
</Paragraph>

<WorksIn visibleTypes={['reactJS', 'html5']} />

<ParameterList
  parameters={[
    {
      name: 'form',
      description: 'The form element, its ID, or null to use the document body.',
      type: 'HTMLFormElement | HTMLDivElement | string | null',
      dataType: 'HTMLElement',
      defaultValue: 'null',
      returnType: 'boolean'
    }
  ]}
/>

<Headings.h5 className="mt-4">Usage</Headings.h5>

<Headings.h6 className="mt-4">Example: Reset Form Elements</Headings.h6>

<CodeBlock>
{`const formId = "myForm";
nfsfu234FormValidation.reset(formId); // Resets the form with ID 'myForm'

const formElement = document.getElementById('myForm');
nfsfu234FormValidation.reset(formElement); // Resets the form element directly
`}
</CodeBlock>
